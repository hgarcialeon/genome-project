# ADR-0001: Specification First

## Status

Accepted

## Context

Genome aims to become the declarative language for autonomous organizations.

If implementation decisions lead the project, Genome risks becoming another agent dashboard rather than a durable standard.

## Decision

Genome will be specification-first.

Major concepts must be defined in RFCs, SPEC files, or ADRs before implementation.

## Consequences

- Development may move slower at first.
- Architecture remains coherent.
- Contributors can understand decisions without chat history.
- The specification can outlive any implementation.
